zustand 스토어 폴더임.

하나의 스토어에다가 다 때려박지 않고,
하나하나 기능별로 소분해서 코딩하였음.

knob_on_off_store.tsx 얘는 들어가보면 state랑 셋 함수가 두 개씩 적혀있는데,
이는 한글 노브와 영문 노브를 구분하기 위함임.

하나하나 설명해보자면,
    1. body_page_store.tsx 
        얘는 몸통 부분에서 대괄호 안에 페이지가 몇번째인지 관리함.
        디폴트가 0이며, 왼쪽 괄호를 누르면 -1씩, 오른쪽 괄호를 누르면 +1씩 바뀜.

    2. hovered_book_store.tsx
        얘는 북리스트에서 선택된 꿈끼깡꾀 매거진이 몇번째인지 관리함.
        북 리스트에 마우스 올리거나 초성 버튼을 눌러서 필터링 되면 이 녀석이 바뀜.
        이게 바뀌면 리스트 좌측에 인디케이터랑 표지 썸네일이 렌더링 됨.

    3. is_arm_animatiog.tsx
        얘는 몸통에서 괄호 클릭시 팔 페이지 전환 애니메이팅 되는 중인지 아닌지 관리함.
        애니메이션이 재생중이면 true, 아니면 false가 됨.
        괄호를 빠르게 클릭 시, 애니메이션이 중첩되어 페이지 렌더가 제대로 되지 않는 문제를 해결하기 위해 작성됨.

    4. knob_on_off_store.tsx
        얘는 몸통 우측에 토글 노브 켜져있는지 아닌지 관리함.
        이게 바뀌면 버튼들과 노브의 애니메이션이 재생됨

    5. letter_logo_store.tsx
        얘는 현재 선택된 초성 로고(버튼 말고 썸네일이랑 리스트 위에 있는 로고)가 무엇인지 관리함.
        ㄱ: 0, ㄲ: 1, ..., z: 45 였을거임
        초성 버튼을 누르면 이게 바뀌고, 이게 바뀌면 로고가 바뀜.

    6. page_direction_store.tsx
        얘는 몸통 내 팔들이 전환 될 때, 애니메이션의 방향을 관리함.
        왼쪽 괄호를 누르면 -1이 되고, 오른쪽 괄호를 누르면 +1이 됨.
        -1이면 페이지가 왼쪽으로 넘어가는 애니메이션이, +1 이면 반대로 렌더됨.
